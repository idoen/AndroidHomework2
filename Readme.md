# Obstacles - Readme

## מה התווסף למשחק
- **מנוע משחק עם מכשולים ומטבעות**: יצירת מכשולים אקראיים בנתיבים שונים, איסוף מטבעות ומעקב אחר מרחק.  
- **לבבות וחיווי התנגשות**: חיווי חיים ב־UI, רטט והתראה קולית בעת התנגשות.  
- **מצבי שליטה**: בחירה בין שליטה באמצעות כפתורים לבין שליטה באמצעות חיישני תנועה.  
- **מסכי תפריט וסיום משחק**: תפריט ראשי לבחירת מצב ושליטה, ומסך סיכום עם נתוני מרחק ומטבעות.  
- **טבלת שיאים + מפה**: שמירת שיאים מקומית והצגה בטבלה, כולל תצוגת מפה עם מיקום השיאים.  
- **שימוש במיקום**: ניסיון לצרף נקודת מיקום לכל שיא (כאשר יש הרשאות).  

## איך מריצים

### דרישות
- Android Studio / Gradle
- SDK מתאים (מוגדר `compileSdk 35`, `minSdk 29`)
- חבילת Google Maps (משתמשת ב־`play-services-maps`)

### הגדרת מפתח API למפות (MAPS_API_KEY)
נדרש מפתח Google Maps כדי להציג את מפות השיאים.

אפשר להגדיר באחת הדרכים:

1. **דרך `gradle.properties` (מומלץ):**
   ```properties
   MAPS_API_KEY=YOUR_KEY_HERE
   ```

2. **או כמשתנה סביבה:**
   ```bash
   export MAPS_API_KEY=YOUR_KEY_HERE
   ```

### הרצה
- פתחו את הפרויקט ב־Android Studio ולחצו **Run**.
- או דרך CLI:
  ```bash
  ./gradlew :app:installDebug
  ```

### הרשאות
האפליקציה תבקש הרשאת מיקום כדי לצרף מיקום לשיאים ולהציגם על המפה.
